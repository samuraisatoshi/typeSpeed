When you click "Select Code Folder", your browser may show a dialog asking about file uploads. This is a standard browser security notice.

Important: Despite what the dialog says, TypeSpeed does NOT upload your files anywhere. The files are only read locally in your browser for typing practice.

To avoid confusion:

  1. You can safely select your code folder - files remain on your computer
  2. For extra peace of mind, use a test folder with sample code
  3. Or run from a local server to avoid the dialog entirely:
    python3 -m http.server 8000 (Python)
    npx http-server -p 8000 (Node.js)
    php -S localhost:8000 (PHP)
