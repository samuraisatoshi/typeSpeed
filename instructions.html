üöÄ Quick Start

Easiest Method: Simply open index.html directly in your browser!
  1. Open index.html in your web browser
  2. Click "Select Code Folder" and choose a folder with source code
  3. Click "Start Typing" to begin practicing
  4. Type the displayed code as accurately as possible

üîç Understanding the Browser Dialog

Important: When selecting a folder, your browser may show a dialog saying "Upload files to this site?" - This is misleading! TypeSpeed does NOT upload your files anywhere.

Why does this happen?

Modern browsers show this security dialog whenever a web page wants to access files on your computer, even if it's just reading them locally. The browser uses the same dialog for both uploading and local reading, which can be confusing.

Is it safe?

Yes, absolutely! TypeSpeed:

  • ‚úÖ Runs 100% offline in your browser
  • ‚úÖ Never sends your files anywhere
  • ‚úÖ Has no server connection capabilities
  • ‚úÖ Cannot access the internet
  • ‚úÖ Only reads files to display them for typing practice

üñ•Ô∏è Server Setup Options

If you prefer to avoid the browser dialog entirely, you can run TypeSpeed from a local server. Here are several options:

Option 1: Node.js (Recommended)

If you have Node.js installed:

npx http-server -p 8000

Or install globally first:

npm install -g http-server
http-server -p 8000

Then open: http://localhost:8000

Option 2: Python (Built-in)

Python is pre-installed on most systems:

Python 3:

python3 -m http.server 8000

Python 2:

python -m SimpleHTTPServer 8000

Then open: http://localhost:8000

Option 3: PHP

If you have PHP installed:

php -S localhost:8000

Then open: http://localhost:8000

Option 4: Ruby

If you have Ruby installed:

ruby -run -e httpd . -p 8000

Then open: http://localhost:8000

Option 5: Visual Studio Code

Using VS Code with Live Server extension:

  1. Install "Live Server" extension
  2. Right-click on index.html
  3. Select "Open with Live Server"

üîß Troubleshooting

Common Issues

Files not loading?

  • Ensure you selected a folder with source code files
  • Check that files have supported extensions (.js, .py, etc.)
  • Try refreshing the page and selecting again

Statistics not saving?

  • Check that localStorage is enabled in your browser
  • Ensure you're not in Private/Incognito mode
  • Some browsers clear localStorage when closed

Keyboard input issues?

  • Click on the code display area to focus
  • Ensure no browser extensions are intercepting keystrokes
  • Try a different browser if issues persist

Browser Compatibility

TypeSpeed works best on:

  • Chrome/Edge 90+
  • Firefox 88+
  • Safari 14+
Note: Older browsers may not support all features. Please update your browser for the best experience.
