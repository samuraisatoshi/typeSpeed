# title: Scan Project for Code Files
# shortDescription: Scan a folder and catalog all source code files for typing practice

This command scans a specified project folder for source code files and catalogs them for typing practice.

## Steps:
1. Use the @code-parser agent to scan the folder
2. Filter files by supported languages
3. Generate metadata for each file
4. Store in the code repository
5. Report statistics

Usage: `/scan-project <folder-path>`
