# title: Start Typing Session
# shortDescription: Begin a new typing practice session with a random code file

This command starts a new typing practice session using a randomly selected code file from the repository.

## Steps:
1. Select a random code file (optionally filtered by language)
2. Use @formatter-specialist to prepare the code
3. Initialize typing session with @metrics-analyst
4. Launch the typing interface
5. Track performance in real-time

Usage: `/start-typing [language]`
